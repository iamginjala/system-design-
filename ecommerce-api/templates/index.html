üìñ About This API

This is a production-ready e-commerce API built with Flask, implementing a dual-database architecture for optimal performance. The API uses PostgreSQL for dynamic transactional data and Redis for caching static data like product catalogs.

The API features JWT-based authentication, role-based authorization (Admin/Customer), and a GraphQL interface for flexible querying.

üîç Example GraphQL Queries

Get All Products (Cached)

{
  getProducts {
    productId
    stockCount
    price
    lastUpdated
  }
}

Get Product by ID

{
  getProductsById(id: "product-uuid-here") {
    productId
    stockCount
    price
    lastUpdated
  }
}

Get Orders (Customer: Own Orders | Admin: All Orders)

{
  getOrders {
    orderId
    customerId
    totalPrice
    status
    createdAt
  }
}

Get Order by ID

{
  getOrderById(orderId: "order-uuid-here") {
    orderId
    customerId
    totalPrice
    status
    createdAt
  }
}

Create Order (Mutation)

mutation {
  createOrder(productId: "product-uuid", quantity: 2) {
    orderId
    totalPrice
    status
  }
}

Password Requirements

  • At least one lowercase letter
  • At least one number
  • At least one special character (@$!%*?&)
